# 🚀 快速开始

## 你现在需要做的

### 1. 下载并安装 Android Studio
下载地址：https://developer.android.com/studio
- 不需要注册账号
- 安装时全部默认选项即可

### 2. 打开项目
在当前目录运行：
```bash
npm run open
```

### 3. 构建 APK
Android Studio 打开后：
1. 等待 Gradle 构建完成（右下角进度条）
2. 点击菜单：Build > Build Bundle(s) / APK(s) > Build APK(s)
3. 等待完成，点击通知中的 locate
4. 找到 `app-debug.apk`

### 4. 安装到手机
把 APK 发到手机，点击安装。
